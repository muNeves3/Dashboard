## Getting Started :computer:

# Make a clone of the repository: :robot:

```sh
git clone https ://github.com/muNeves3/Dashboard
```

# Go to the directory ↪️

```sh
cd Dashboard
```

# Install the dependencies: :rocket:

```sh
npm install
```

# Run the app :rocket:

```sh
yarn start
```
